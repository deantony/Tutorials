Go To "Command Prompt"
Check Version of Python (2.x or 3.x)

type:
python -v

determine version of python
then type:
ctrl-z then enter

Pip Install Module

if python 3.x:
type:
python -m pip install <module_name>
i.e:
python -m pip install pandas

if python 2.x:
type:
python3 -m pip install <module_name>
i.e:
python3 -m pip isntall pandas
